# LMS Landing Page

A pixel-perfect mock of an LMS (Learning Management System) landing page featuring:

- **All Courses Dashboard** — Course grid with filter tabs, progress tracking, and category filters
- **Course Inside View** — Three-panel learning experience with sidebar navigation, video lesson player, and AI Playground practice area

## Getting Started

Open `index.html` in your browser.

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Inter)
- Font Awesome 6 Icons
